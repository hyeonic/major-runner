# Major Runner

 모든 전공인들이 함께 달려가는 공간, Major Runner
 
같은 꿈을 안고 있는 대학생, 취업 준비생들이 모여 정보를 공유하고, 스터디를 모집하며, 질의응답 등을 나눌 수 있는 공간입니다.   

## 사용 기술
 - vue.js 2.6.11
 - Vue Router 3.4.3
 - Vuex 3.5.1
 - Axios 0.20.0
 - fontawsome 5.14.0
